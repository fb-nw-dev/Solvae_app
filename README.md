# Aplicativo de Prestação de Serviços (B2B / C2C)

Projeto desenvolvido como parte do curso técnico, focado em facilitar a prestação de serviços e a conexão entre pessoas físicas e empresas.

## Status do Projeto
O aplicativo está atualmente em fase de desenvolvimento. A interface e a estrutura de telas já foram totalmente implementadas, e o próximo passo é o desenvolvimento da lógica de dados e regras de negócio.

## Tecnologias Utilizadas
*   **Linguagem:** Kotlin
*   **Plataforma:** Android

## Objetivos de Aprendizado
Este projeto está sendo usado para consolidar meus conhecimentos em:
*   Criação de interfaces no Android.
*   Navegação entre telas.
*   Futura implementação de estruturas de dados (como Maps e listas).
*   Uso do retrofit
*   costrução de um aplicativo real
*   Integração com APIs usando **Retrofit**.
*   Construção e arquitetura de um aplicativo real focado no mercado.
